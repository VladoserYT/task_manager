# Task Manager (Менеджер задач)

Простое desktop-приложение для управления задачами, разработанное на Python с использованием библиотеки Tkinter для графического интерфейса.

## Функциональность

- **Добавление задач:** Ввод названия, описания и срока выполнения.
- **Удаление задач:** Удаление выбранной задачи из списка.
- **Автосохранение:** Все задачи автоматически сохраняются в файл `tasks.json` и загружаются при следующем запуске.
- **ООП:** Использование классов `Task` и `TaskManager` для структурирования кода.

## Запуск приложения

Для запуска приложения вам потребуется установленный **Python 3.x**.

1.  **Сохраните файлы:** Убедитесь, что файлы `classes.py` и `gui.py` находятся в одной папке.
2.  **Запустите:** Откройте терминал (командную строку) в этой папке и выполните команду:

    ```bash
    python3 gui.py
    ```

    *(Если команда `python3` не работает, попробуйте `python`)*

3.  **Использование:**
    *   Введите **Название**, **Описание** и **Срок выполнения** (в формате ГГГГ-ММ-ДД, например, 2025-12-31).
    *   Нажмите **"Добавить задачу"**.
    *   Чтобы удалить задачу, выберите ее в списке и нажмите **"Удалить выбранную"**.
    *   Файл `tasks.json` будет создан автоматически для сохранения ваших задач.

## Структура проекта

- `classes.py`: Содержит логику приложения (классы `Task` и `TaskManager`).
- `gui.py`: Содержит графический интерфейс (Tkinter) и основную логику запуска.
- `tasks.json`: Файл, в котором хранятся все ваши задачи в формате JSON.
- `README.md`: Этот файл с описанием проекта.
